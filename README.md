# egg-born-module-cabloy-cmsthemecommunityenus

Cabloy-CMS Theme

## Installation

``` bash
$ npm i egg-born-module-cabloy-cmsthemecommunityenus
```

## Usage

Changing languange's theme to `cabloy-cmsthemecommunityenus`

``` javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cabloy-cmsthemecommunityenus"
}
```
